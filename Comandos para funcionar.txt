PYTHON III - DJANGO

- Criar o env:
    ```
    py -m venv env
    ```
- Habilitar o env
    ```
    env\Scripts\activate
    ```
- Inslatar o Django
    ```
    pip install Django
    ```
- Gerar o arquivo de requirements.txt
    ```
    pip freeze > requirements.txt
    ```
- Criar o projeto
    ```
    django-admin startproject setup .
    ```
- Acessar o help do Django
    ```
    py manage.py --help
    ```
